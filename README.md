# Customer Purchase Insights

## Overview
This project analyzes customer shopping behavior using transactional retail data to uncover spending patterns, customer segments, and product performance. It demonstrates an end-to-end analytics workflow using Python, PostgreSQL, and Power BI.

## Business Objective
Help a retail business understand how demographics, discounts, purchase frequency, and subscriptions influence customer behavior and revenue to support data-driven decision-making.

## Dataset
- 3,900 retail transactions  
- Customer demographics, purchase details, and behavioral features  

## Technical Workflow
- Python-based ETL for data cleaning and feature engineering  
- PostgreSQL for analytical SQL queries
- Excel for quick exploration, pivot tables, and validation  
- Power BI for interactive dashboards and visualization  

## Deliverables
- Python notebook for EDA and ETL  
- SQL scripts for analytical queries
- Excel workbook for intermediate analysis and pivot tables
- Power BI dashboard  
- Final report and presentation  

## Tools & Technologies
- Python (Pandas)  
- PostgreSQL
- Excel  
- SQL  
- Power BI

## How to Run This Project

### 1. Clone the repository
Run the following commands in your terminal:
```bash
git clone https://github.com/Travis-Shakeel/Customer-Purchase-Insights
cd Customer_Purchase_Insights
```


### 2. Open the Python Notebook
- Navigate to the `python/` folder  
- Open `Customer_Purchase_Insights_Analysis.ipynb` using Jupyter Notebook or JupyterLab  

The notebook includes:
- Data loading and exploration  
- Data cleaning and feature engineering  
- Preparation for database analysis  

### 3. Set Up PostgreSQL
- Create a local PostgreSQL database  
- Update the database credentials in the notebook before running the connection cell  

### 4. Run the Notebook
- Execute the cells sequentially to transform the raw data  
- Load the cleaned dataset into PostgreSQL  

### 5. Run SQL Analysis
- Open `sql/Customer_Purchase_Insights_SQL.sql`  
- Execute the queries against the PostgreSQL database to answer business questions  

### 6. Review Insights and Visuals
- Power BI dashboard screenshots are included in the project report  
- Key insights and recommendations are summarized in the final report  

### Key Takeaways
- Demonstrates an end-to-end data analysis and ETL workflow  
- Highlights the ability to combine Python, Excel, SQL, and BI tools  
- Focuses on translating data insights into actionable business recommendations
